# PetTracker

Приложение для отслеживания перемещения домашних животных (кошек и собак) с использованием трекеров. Позволяет сохранять данные о нахождении животных в зоне или вне зоны энергосбережения и запрашивать их через REST API.

---

## Описание

- Приложение на **Java 21** с **Spring Boot 3+**.
- Сборка приложения с помощью Maven
- REST API для передачи данных с трекеров и получения сохранённых данных.
- Встроенная поддержка in-memory базы данных **H2**, легко заменяемой на другую.
- Поддержка двух типов трекеров для кошек (small, big) и трёх для собак (small, medium, big).
- Включены модульные и интеграционные тесты.
- Swagger/OpenAPI для документации API.

---
## Настройка профилей БД
Для смены БД необходимо в файле compose.yml указать нужный профиль
1. По умолчанию стоит `SPRING_PROFILES_ACTIVE=h2`
2. Для переключения на PostgreSQL нужно указать `SPRING_PROFILES_ACTIVE=postgres` а так же настроить коннект в файле `application-postgres.yml`

## Тестирование проекта
1. Склонировать проект из репозитория
2. Запустить из корневой директории проекта докер командой `mvnw test`


## Развертывание в Docker
1. Склонировать проект из репозитория
2. Запустить из корневой директории проекта докер командой `docker-compose up -d --build`
3. После запуска все енд-пойнты будут доступны по адресу http://localhost:8080/


## Енд-пойнты для взаимодействия с REST API
1. Описание форматов запросов/ответов `swagger-ui/index.html` 
2. Создание новых записей животных `POST:  /api/v1/pets`
3. Обновление статуса трекера `PUT: api/v1/pets/{UUID}`
4. Получение информации о животных, находящихся за пределами зоны `GET: /api/v1/pets/out-of-zone`
